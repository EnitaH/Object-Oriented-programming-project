# 🕹️ Object-Oriented Adventure Game
A **Java console adventure game** built to demonstrate **Object-Oriented Programming principles and software design patterns**.

Players explore rooms, encounter enemies, choose themes, and use different weapons and combat strategies.

This project was developed as part of an **Object-Oriented Software Design coursework**.

---

# 🎮 Game Overview

The player navigates through multiple rooms where they must:

- fight enemies
- solve challenges
- use weapons
- apply different combat strategies

The system is designed using **modular OOP architecture**, making it easy to extend with new rooms, enemies, or weapons.

---

# ✨ Features

- 🧟 Multiple enemy types (Zombie, Magical Creature)
- ⚔️ Different weapons (Knife, Sword, Golden Sword)
- 🏠 Multiple room types
- 🎯 Combat strategy system
- 🎭 Theme selection system
- 🧩 Puzzle helper
- 🏗 Modular design using design patterns

---

# 🧠 Design Patterns Used

### Command Pattern
Used for executing player actions.

Examples:
- `EnterRoomCommand`
- `ChooseThemeCommand`

---

### Strategy Pattern
Allows enemies to use different fighting behaviours.

Examples:
- `FancyFightStrategy`
- `ZombieFightStrategy`

---

### Factory Pattern
Used to dynamically create game objects.

Examples:
- `RoomFactory`
- `WeaponFactory`

---

# 🧱 Object-Oriented Concepts

This project demonstrates:

- **Encapsulation**
- **Inheritance**
- **Polymorphism**
- **Abstraction**

These concepts help create a **clean, maintainable architecture**.
